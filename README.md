This repository contains all project files for a research project on healthcare analytics, focused on building an interactive Power BI dashboard for efficient hospital resource management.

The dashboard provides visual insights into patient admissions, staff distribution, bed occupancy, emergency cases, and departmental performance. It aims to support data-driven decision-making and enhance the overall hospital operational efficiency.

Repository Contents:

date_data.csv – Time-based dataset used for trend and timeline analysis.

Hospital ER_Data.csv – Main dataset containing hospital emergency and resource data.

HRM_Logo.png – Custom logo designed for the dashboard interface.

RP_HRM_dashboard.pbix – Power BI project file containing all visualizations and measures.

RP_HRM_dashboard.pdf – Exported report version for documentation and presentation.

This project demonstrates the integration of data visualization, healthcare analytics, and resource management using Power BI as part of an academic research initiative.
